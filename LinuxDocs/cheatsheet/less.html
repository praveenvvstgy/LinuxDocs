less

Opens a file for reading Allows movement and search Doesn't read the entire file (suitable for logs)

  • open a file 

less {{source_file}}

  • page up / down 

d (next), D (previous)

  • go to start / end of file 

g (start), G (end)

  • search for a string 

/{{something}}   then   n (next), N (previous)

  • exit 

q
