Name

setsid — run a program in a new session

Synopsis

setsid program [ arg... ]

DESCRIPTION

setsid runs a program in a new session.

OPTIONS

−c , −−ctty

Set the controlling terminal to the current one.

−w , −−wait

Wait the execution of the program to end, and return the exit value of the child as return value of the setsid .

SEE ALSO

setsid (2)

AUTHOR

Rick Sladkey <jrs@world.std.com>

AVAILABILITY
