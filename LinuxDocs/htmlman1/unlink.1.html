Name

unlink — call the unlink function to remove the specified file

Synopsis

unlink FILE

unlink OPTION

DESCRIPTION

Call the unlink function to remove the specified FILE.

−−help

display this help and exit

−−version

output version information and exit

GNU coreutils online help: < http://www.gnu.org/software/coreutils/ > Report unlink translation bugs to < http://translationproject.org/team/ >
