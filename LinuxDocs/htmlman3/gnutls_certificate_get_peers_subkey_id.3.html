Name

gnutls_certificate_get_peers_subkey_id — API function

Synopsis 

#include <gnutls/gnutls.h>
int gnutls_certificate_get_peers_subkey_id ( gnutls_session_t session ,
  gnutls_datum_t * id ) ;
 

ARGUMENTS

gnutls_session_t session

is a gnutls session

gnutls_datum_t * id

will contain the ID

DESCRIPTION

Get the peer's subkey ID when OpenPGP certificates are used. The returned id should be treated as constant.

RETURNS

On success, GNUTLS_E_SUCCESS (0) is returned, otherwise an error code is returned.

SINCE

3.1.3
