Name

gnutls_digest_get_name — API function

Synopsis 

#include <gnutls/gnutls.h>
const char * gnutls_digest_get_name ( gnutls_digest_algorithm_t algorithm ) ;
 

ARGUMENTS

gnutls_digest_algorithm_t algorithm

is a digest algorithm

DESCRIPTION

Convert a gnutls_digest_algorithm_t value to a string.

RETURNS

a string that contains the name of the specified digest algorithm, or NULL .
