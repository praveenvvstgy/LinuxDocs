Name

strlen — calculate the length of a string

Synopsis 

#include <string.h>
size_t strlen ( const char * s ) ;
 

DESCRIPTION

The strlen () function calculates the length of the string s , excluding the terminating null byte ('\0').

RETURN VALUE

The strlen () function returns the number of bytes in the string s .

ATTRIBUTES

Multithreading (see pthreads(7))

The strlen () function is thread-safe.

CONFORMING TO

SVr4, 4.3BSD, C89, C99.

SEE ALSO

string (3) , strnlen (3) , wcslen (3) , wcsnlen (3)
