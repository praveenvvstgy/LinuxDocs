Name

gnutls_priority_kx_list — API function

Synopsis 

#include <gnutls/gnutls.h>
int gnutls_priority_kx_list ( gnutls_priority_t pcache ,
  const unsigned int ** list ) ;
 

ARGUMENTS

gnutls_priority_t pcache

is a gnutls_prioritity_t structure.

const unsigned int ** list

will point to an integer list

DESCRIPTION

Get a list of available key exchange methods in the priority structure.

RETURNS

the number of curves, or an error code.

SINCE

3.2.3
