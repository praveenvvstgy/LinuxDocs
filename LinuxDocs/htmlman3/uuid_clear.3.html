Name

uuid_clear — reset value of UUID variable to the NULL value

Synopsis 

#include <uuid.h>
void uuid_clear ( uuid_t uu ) ;
 

DESCRIPTION

The uuid_clear function sets the value of the supplied uuid variable uu to the NULL value.

AUTHOR

Theodore Y. Ts'o

AVAILABILITY

libuuid is part of the util-linux package since version 2.15.1 and is available from ftp://ftp.kernel.org/pub/linux/utils/util-linux/.
